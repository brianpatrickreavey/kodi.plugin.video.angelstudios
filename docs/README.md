# Documentation

This directory contains documentation for the Kodi Angel Studios plugin.

## Directory Structure

- **`dev/`** - Developer-focused technical documentation
  - Implementation details, data structures, performance considerations
  - API mappings, caching strategies, testing guidelines
  - Workflow and development practices

- **`archive/`** - Completed plans, research, and historical documentation
  - Archived optimization plans and research documents
  - Historical implementation details no longer active

- **`features/`** - Feature documentation and specifications
  - Planned features, requirements, and design documents
